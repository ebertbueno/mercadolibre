# mercadolibre
Mercado Libre API for laravel

# Install

Run composer requre:

```shell
$ composer require javiertelioz/mercadolibre
```

Add to config/app.php the service provider

```php
    'providers' => [

        /**
         * Other Providers...
         */
        Javiertelioz\MercadoLibre\MeliServiceProvider::class,
    ],
```
